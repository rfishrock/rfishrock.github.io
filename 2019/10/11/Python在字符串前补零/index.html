Python在字符串前补零

发表于 2019-10-11 分类于 Python 阅读次数： Valine：

用于对不规则字符串（如命名）进行转换。

zfill() & rjust()

Python zfill() 方法返回指定长度的字符串，原字符串右对齐，前面填充0。

Python rjust() 返回一个原字符串右对齐,并使用空格填充至长度 width 的新字符串。如果指定的长度小于字符串的长度则返回原字符串。

  • width – 指定填充指定字符后中字符串的总长度.
  • fillchar – 填充的字符，默认为空格。
